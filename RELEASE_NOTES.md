# Release v1.0.0

Первая публичная версия `mafile renamer + cut`.

## Что есть в релизе

- графическое Windows-приложение на `tkinter`
- обработка всех `.mafile` в выбранной папке
- извлечение `shared_secret`, `Session.SteamID` и `account_name`
- автоматическое переименование файлов по `account_name`
- сохранение очищенных файлов в папку `ready`
- лог обработки с количеством успешных и пропущенных файлов

## Для GitHub Release

Тег:

v1.0.0
```

- сохранение только shared_secret и Session.SteamID
- автоматическое создание папки ready
- простой графический интерфейс для Windows
```
